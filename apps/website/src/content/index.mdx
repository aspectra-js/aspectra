# Introduction

The reccomended way to run aspectra is using [`tsx`](https://github.com/privatenumber/tsx).

### Installation

```sh npm2yarn
npm i aspectra
```

Then, add the following to your `tsconfig.json`:

```json
{
  "compilerOptions": {
    "strict": true,
    "target": "es2022", // or higher
    "experimentalDecorators": false // or remove this line
  }
}
```
