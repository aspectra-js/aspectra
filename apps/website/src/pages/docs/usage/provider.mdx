# @provider

`providers` are classes that provide a service to the application. An example
of a `provider` is a database connection:

```typescript filename="src/providers/database.ts"
import { provider } from 'aspectra'

@provider
export class Database {
  public connect() {
    console.log('Connecting to the database')
  }

  public getAll() {
    return ['item1', 'item2']
  }

  public close() {
    console.log('Closing the database connection')
  }
}
```
